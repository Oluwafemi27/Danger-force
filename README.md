# Free Fire MAX - Android Project

This repository contains the extracted source files from an Android application package (APK) for Free Fire MAX, a popular mobile game.

## Project Structure

- **AndroidManifest.xml** - Android application manifest (binary format)
- **assets/** - Application assets including:
  - Google Services configuration
  - Unity game engine data
  - Managed code and resources
- **kotlin/** - Kotlin standard library files
- **res/** - Android resources (layouts, drawables, animations, etc.)
- **META-INF/** - Metadata information
- **resources.arsc** - Compiled Android resources

## Key Components

- **Package Name**: `com.dts.freefireadv`
- **Firebase Project**: free-fire-8cd39
- **Game Engine**: Unity3D
- **Platform**: Android

## Resources

The `res/` directory contains various Android resources organized by type and screen density:
- Drawable resources for different screen densities (ldpi, mdpi, hdpi, xhdpi, xxhdpi, xxxhdpi)
- Layout files for different Android versions
- Animation and interpolator files
- Color definitions

## Note

This appears to be a decompiled/extracted Android application. The AndroidManifest.xml and other files are in binary format as they would appear in a compiled APK file.
